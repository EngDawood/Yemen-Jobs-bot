# Project Brief: RSS-ify

## Core Mission
RSS-ify is a lightweight, NodeJS-powered Telegram bot designed to keep users updated by delivering content from their favorite RSS feeds directly to a Telegram group chat or private message.

## Key Features
- **RSS Feed Subscriptions**: Users can add, remove, and list their RSS feed subscriptions.
- **Direct Delivery**: Delivers new content from feeds directly to Telegram.
- **Customization & Management**: Provides commands to manage subscriptions effortlessly.
- **Group & Private Chats**: Works in both group chats and private conversations.

## Deployment
The bot can be self-hosted. The deployment process involves:
1. Cloning the repository with submodules.
2. Running a `start.sh` script with different flags for clean or dirty runs.
