## Deployment
```
git clone --recurse-submodules https://github.com/EngDawood/Yemen-Jobs-bot.git
cd Yemen-Jobs-bot
```

clean start with git pull
```
bash start.sh -c 
```
dirty run with git pull
```
bash start.sh -d 
```
start Yemen-Jobs-bot directly
```
bash  start.sh
```
